## ソラニワの拡張機能

| ファイル名 | 説明 |
|---|---|
| `actionlog.user.js` | **戦闘ログ**：キャラの顔アイコンをクリックすると、そのキャラの行動ログが見れます。 |
| `add5btn.user.js` | **各種行動**：ボタンを色々と追加。 |
| `kadanpicker.user.js` | **花壇**：植える花を選ぶ際、プルダウンじゃなくて花図鑑から選べるようにします。 |
| `mapguide.user.js` | **マップ移動**：カーソルを合わせると、その座標を良い感じに表示。 |
| `skillmix.user.js` | **スキル合成**：合成時にプレビュー画面を追加。 |
| `tsv.user.js` | **キャラリスト**：閲覧時、右の方にステータス合計値を追加。 |
